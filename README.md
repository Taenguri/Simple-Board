# Simple-Board

간단한 회원제 게시판 만들기

프로젝트 이름 : Simple Board
개발 인원 :  1명
개발 기간 :  2022.01.10 ~ 20.02.01
주요 기능  -  회원가입 및 로그인
    -  게시글 등록, 수정, 삭제		
    -  페이징 및 검색기능 
    -  책 상세보기
    -  댓글 작성, 수정, 삭제 기능
개발 환경  :  Spring framework, Java,, HTML5,CSS3, Jquery, oracle,  bootstrap

간단 소개  :  회원제 게시판(CRUD) 입니다.
